## 🚀 **COMANDOS DE DESARROLLO**

### **Instalación:**
```bash
# Instalar dependencias
npm install

# iOS (requiere Xcode)
cd ios && pod install && cd ..

# Android (requiere Android Studio)
npx react-native run-android

# Metro bundler
npx react-native start --reset-cache
```

### **Testing:**
```bash
# Unit tests
npm test

# E2E tests
npm run test:e2e

# Linting
npm run lint

# Build para producción
npm run build:android
npm run build:ios
```
